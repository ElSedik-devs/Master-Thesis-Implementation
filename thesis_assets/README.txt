
Thesis assets export directory
=============================

tables/
  CSV tables for metrics, per-image results, per-instance results, comparisons

  Main thesis-ready plots:
  - threshold analysis
  - robustness plots
  - baseline vs fine-tuned mobile plots

  clean_test/
  mobile_baseline/
  mobile_finetuned/
  corruptions/

Each examples folder contains many saved detection figures:
- best_tp_*.png
- worst_fn_*.png
- fp_heavy_*.png

checkpoint_reports/
  Selected checkpoint paths and related notes
